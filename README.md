# 🍺 Cerveza 722 — Sitio Web Oficial

Sitio web estático para **Cerveza 722**, una cervecería artesanal asturiana fundada en 1998 en Llanes (Asturias). Elaborada con agua de los Picos de Europa y malta seleccionada, con más de 25 años de tradición cervecera.

---

## 📋 Descripción

Este proyecto es el sitio web corporativo de Cerveza 722. Consta de tres páginas principales que presentan la marca, el catálogo de cervezas y la historia de la empresa. Desarrollado íntegramente con **HTML5** y **CSS3** puro, sin dependencias externas ni frameworks.

## 📄 Páginas

| Página | Descripción |
|---|---|
| `index.html` | Inicio: hero con logo, valores de la marca y sección doble con enlace a la historia |
| `nuestras-cervezas.html` | Catálogo con las 6 variedades de cerveza, estilo y graduación |
| `nuestra-historia.html` | Línea de tiempo desde 1998 hasta hoy |

## 🚀 Cómo ejecutar el proyecto

Al tratarse de un sitio estático no requiere instalación ni servidor. Basta con abrir el archivo `index.html` directamente en el navegador o usar la extensión **Live Server** de VS Code para un entorno de desarrollo más cómodo.

```bash
# Con Live Server (VS Code)
# Clic derecho sobre index.html → "Open with Live Server"
```
