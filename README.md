# Web Cerveza 722 

Este repositorio contiene el sitio web de **Cerveza 722**, una marca ficticia de cerveza artesanal elaborada en los Picos de Europa (Asturias).

El objetivo del proyecto ha sido crear una web moderna, visual y totalmente responsive utilizando únicamente estándares web, sin depender de librerías externas.

##  Tecnologías utilizadas

*   **HTML5:** Estructura semántica (header, nav, section, footer).
*   **CSS3:** Estilos personalizados desde cero.
    *   Uso de **Flexbox** para la cabecera y el pie de página.
    *   Uso de **CSS Grid** para el catálogo de productos y la sección de valores.
    *   Variables CSS para mantener la coherencia en los colores (Azul Picos, Naranja Atardecer).
    *   Diseño **Responsive** adaptado a móviles y escritorio.

##  Estructura del sitio

El proyecto consta de tres páginas principales:

1.  **Inicio (`index.html`):** Página de aterrizaje con un banner a pantalla completa, presentación de la marca y los valores fundamentales (agua de montaña, malta seleccionada, etc.).
2.  **Nuestras Cervezas (`nuestras-cervezas.html`):** Catálogo de productos donde se muestran las 6 variedades de cerveza en un diseño de tarjetas (cards).
3.  **Nuestra Historia (`nuestra-historia.html`):** Una línea de tiempo (timeline) vertical que narra la evolución de la empresa desde 1998 hasta hoy.

##  Cómo ver el proyecto

No es necesaria ninguna instalación compleja.

1.  Descarga el código o clona este repositorio.
2.  Abre el archivo `index.html` en tu navegador web favorito (Chrome, Firefox, Edge, etc.).

##  Notas de diseño

Se ha buscado una estética "Full Width" (ancho completo) para dar protagonismo a las imágenes de los paisajes asturianos y de las botellas. La tipografía del logotipo busca transmitir fuerza y tradición, mientras que la navegación es sencilla e intuitiva.
